Plugin Woocommerce connect Goship to shipping

== Description ==
WooGoship help connect to Goship easy

= FEATURES =
* connect to Goship
* add button send to goship in orders page
* manager goship shipping order
* allow customer select shipping method from goship on checkout page

= ROAD MAP =
In the future, this plugin will add more and more features for Woocommerce

= WHERE CAN I CONTRIBUTE MY CODE OR IDEA? =

* You can report bugs or contribute code on [this GitHub repo](https://github.com/KingDarkness/woo-goship).
* Please also do let us know if the "bug" is just a grammar/spelling error in both English and Vietnamese. We try to make our products as perfect as possible.

= INSTALLATION =
1. Upload the plugin files to the `/wp-content/plugins/woo-goship` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Go to the `WooCommerce -> Goship -> Cài đặt` screen and configure the plugin.
